T - type of a variable(Integer, Double, String, Character, and so on)
```java
// creating a stack(מחסנית). - LIFO(Last In First Out)
Stack<T> sta = new Stack<T>();

// putting value into a stack.
void push(T x);
sta.push(x); // example

// getting a value from the stack and removing it from it.
T pop();
T value = sta.pop();

// checking a value from the stack.
T top();
T value = sta.top();

// check if the stack is empty
boolean isEmpty();
if (!sta.isEmpty()) sta.pop();

// a string
String toString() -> "[values from head to tail(0 is head, length-1 is tail)]"
System.out.println(sta);
```

פעולה שמחזירה את גודל המחסנית.
```java
public static int stackSize(Stack<T> stack)
{
	Stack<T> temp = new Stack<T>();
	int counter = 0;

	// moving all the elements to the new stack and counting them
	while(!stack.isEmpty())
	{
		temp.push(stack.pop());
		counter++;
	}

	// moving all of the elements back to the original stack
	while(!temp.isEmpty())
		stack.push(temp.pop());

	return counter;
}
```


פעולת העתקה למחסנית
```java
public static Stack<T> copyStack(Stack<T> stack)
{
	Stack<T> newStack = new Stack<T>();
	Stack<T> temp = new Stack<T>();

	// copying the stack to temp
	while(!stack.isEmpty())
		temp.push(stack.pop());

	// copying back to the stack and the new stack
	while(!temp.isEmpty())
	{
		stack.push(stack.top());
		newStack.push(stack.pop());
	}

	return newStack;
}
```
