# דרך הפתרון
## 1. מה רוצים?
לפני שנקרא את הפסקה הנתונה, קודם כל נקרא את השאלה.

## 2. קריאה
נקרא את הפסקה ונסמן מושגי מפתח המתקשרים לשאלה.

## 3. עיבוד
נפרש את הכתוב במילים שלנו.

## 4. אלימינציה
נעבור לתשובות ונפסול את אלו שאינן מתאימות.
